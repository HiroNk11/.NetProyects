Ejemplos simples de patrones de diseno aplicados a funcionalidades concretas.

1. `01-Strategy-Descuentos` - Cambiar la forma de calcular descuentos sin modificar la calculadora.
2. `02-Factory-MetodosPago` - Crear distintos metodos de pago desde una opcion del usuario.
