# Proyectos de consola

Ejercicios progresivos para practicar fundamentos de C# y programacion orientada a objetos desde aplicaciones de consola.

## Proyectos

1. `01-CalculadoraBasica` - Operaciones matematicas simples.
2. `02-ConversorTemperatura` - Conversion entre unidades de temperatura.
3. `03-AdivinarNumero` - Juego con numeros aleatorios.
4. `04-GestorTareas` - Alta y listado de tareas.
5. `05-SistemaAlumnos` - Gestion simple de alumnos y promedios.
6. `06-AgendaContactos` - Contactos en memoria.
7. `07-AgendaContactosCsv` - Contactos persistidos en archivo CSV.
8. `08-GestorProductos` - Productos, precios y stock.
9. `09-CajeroAutomatico` - Cuenta bancaria, saldo y movimientos.

## Conceptos principales

- Sintaxis basica
- Condicionales y ciclos
- Metodos
- Clases y objetos
- Listas
- Validaciones
- Manejo de archivos
