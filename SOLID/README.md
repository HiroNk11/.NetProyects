# Principios SOLID

Proyectos pensados para mostrar buenas practicas de diseno orientado a objetos.

## Proyectos

1. `01-NotificadorPedidos` - Inversion de dependencias usando una interfaz `INotificador`.
2. `02-GeneradorReportes` - Separacion de responsabilidades para generar y exportar reportes.

## Principios trabajados

- SRP: responsabilidad unica
- OCP: abierto/cerrado
- LSP: sustitucion de Liskov
- ISP: segregacion de interfaces
- DIP: inversion de dependencias
